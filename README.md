# Bean-Life-Cycle-In-Spring

A simple Spring Boot project to **demonstrate the full Bean Lifecycle** in Spring Framework, from creation to destruction.

---

## 🔹 Features

This project showcases:

- **Bean Creation** – Constructor execution
- **Dependency Injection** – Automatic wiring of beans
- **Aware Interfaces** – `BeanNameAware`, `BeanFactoryAware`, `ApplicationContextAware`
- **BeanPostProcessor** – Pre and Post initialization hooks
- **Initialization** – `@PostConstruct`, `InitializingBean`, custom init methods
- **Bean Usage** – Beans ready to serve requests via a Controller
- **Destruction** – `@PreDestroy`, `DisposableBean`, custom destroy methods
